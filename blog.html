Question-1: What are the different ways to select an element in the DOM?

There are four different ways to select an element: getElementById, querySelector, getElementsByClassName, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The key diffrences between them are innerHTML: sets the HTML inside an element, innerText:sets the visible text inside an element and textContent sets all text content, icluding hidden elements.

Question-3: What is event delegation in the DOM?

It is a method where a single parent element handles event for mutiple child elements.Basically event listeners are attached to the parent instead of attaching them to each child element.

Question-4: What is event bubbling in the DOM?

It is a technique where an event begins from the targeted child element and it generates to its parent triggering event handlers along the way.

Question-5: How do you create, add, and remove elements using JavaScript?

There are many methods to add, create and remove elemets such as to create we use document.createElement, to add element we use appendChild(), append(), before(),after().Last but not the least to remove element removeChild().
